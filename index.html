Build Your Own World:

2D tile-based world where users are able to move around,

collect coins and enter a special door to a mini-game or

exit using the yellow door. Users are also able to save

and load existing saved game files. Worlds are made reandomly by seeds

each seed represent a different world where coins are randomly placed.
